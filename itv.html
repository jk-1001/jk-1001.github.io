Easier Way ITV

  • Website: https://easierwayitv.com
  • Frontend: HTML, CSS, Javascript, Bootstrap
  • Backend: PHP, Laravel, MySQL

This is a small business website; its primary function is simply to inform the users of services and prices. There is a simple google map showing the area covered and an easy to complete contact form for potential customers to fill.

The website has a bespoke CRM solution for the client to manage their client's data, vehicles, important documents and any upcoming appointments.

The client wanted the CRM to feel like using Excel, as such their data is displayed and saved asynchronously so it feels like using spreadsheet software. The CRM was also built to work on mobile so the client can easily access it from anywhere to update their database.

The client was very happy with how much it improved their day-to-day workflow.
